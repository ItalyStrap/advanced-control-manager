Now you can easily test your website adding Lorem Ipsum text to your posts by one click.

== Description ==

When you are developing the website, you need fill the post content with some text to format correctly your CSS and make the user experience better, so you can use this plugin to add Lorem Ipsum blocks into your posts contents and don't waste time looking for some text to copy and paste.

By using this plugin you can put Lorem Ipsum automaticaly inside your post content by using only one click.


== Installation ==

1. Upload `lorem-ipsum-for-wp-editor/lorem-ipsum-for-wp-editor.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
